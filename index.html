jOmglook

Simple jquery plugin to show basic social media buttons with a a small javascript snippet. I found a lot of solutions for this for wordpress, but none were quite quick enough for me.

How to use

Simply include the script from wherever your js is hosted. 

<script src="js/jOmglook.min.js"></script>

Then attach jOmglook to a containing div. You can style the containing div however you want!

<div class="share"></div>
<script>
    $('#share').jOmglook();
</script>
